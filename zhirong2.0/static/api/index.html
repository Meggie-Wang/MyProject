1.     概述

1.1        软件简介

       软件基因恶意代码检测引擎软件是上海戎磐网络科技有限公司使用动态沙箱技术，独立研发的一款对软件或者样本本身进行一个动态行为的提取，并进行恶意性判断的动态检测引擎软件。该发明主要进行软件的恶意性判定，并给出详细的行为信息以及基因图与顺序图应用于信息安全领域，保护计算机免受恶意软件的攻击。

3.2.5      下载样本生成的基因文件和基因图片

下载样本生成的基因文件和基因图片( 包括png 图片和svg 图片) ，返回url 路径，调用路径即可完成下载。

（1 ）接口属性

属性

属性值

说明

URL

api/gene_file_download/

下载样本生成的基因文件和基因图片

HTTP 请求方法

GET

返回值格式

json

是否需要验证

是

Token 认证

（2 ）API 参数

参数名称

参数说明

格式

id

检测样本的id, 数字（可在检测历史中获取）

字符串

token

用户登录后返回的凭证口令

字符串

（3 ）使用方法和返回结果

使用方法：

http://<ho st>/api/gene_file_download/?id=XXX&token=XXX
