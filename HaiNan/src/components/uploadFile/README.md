# UploadFile

- 参数
    + uploadApi
        * api.js 中定义的接口字典
    + multiple
        * 多文件上传
        * String: false, true
    + webkitdirectory
        * 文件夹上传
        * String: false, true
    + @progress
        * 上传进度
    + @callback
        * 接口response
    + btnName 按钮显示名称
    + btnImg 按钮显示图片
    + params Object类型，提交的附加参数
